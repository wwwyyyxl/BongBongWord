# BongBongWord 项目结构与功能设计

更新时间：2026-05-25

## 1. 项目定位

`BongBongWord` 是面向 HarmonyOS 6.0.2(22) 的 ArkTS / ArkUI 单词学习应用，当前 App 显示名为 `蹦蹦单词`。工程采用 `products + features + commons` 分层，当前阶段聚焦 V2 第一阶段首页与主 Tab 骨架。

```text
BongBongWord/
|-- AppScope/                  # 应用级配置、应用名和全局分层图标资源
|-- commons/
|   `-- basic/                 # 公共 HSP：主题 token、基础组件、通用工具
|-- features/
|   |-- home/                  # 首页 HSP：首页仪表盘、首页组件、首页资源
|   |-- login/                 # 登录 HSP：当前为模板占位
|   |-- mine/                  # 我的 HSP：当前为模板占位
|   |-- pet/                   # 词灵伙伴 HSP：当前为模板占位，已有 Pet 方案文档
|   `-- practice/              # 练习 HSP：当前为模板占位
|-- products/
|   `-- bongbongword/          # Entry HAP：应用入口、Ability、主 Tab 容器
|-- hvigor/                    # Hvigor 工程配置
|-- build-profile.json5        # 工程模块与 SDK 配置
|-- oh-package.json5           # 根依赖声明
|-- PET_SHAPE_PLAN.md          # Lexi / Pet 形态方案
`-- docs/
    `-- 项目结构与功能设计.md
```

## 2. 关键目录与文件注释

### 2.1 AppScope

```text
AppScope/
|-- app.json5
`-- resources/base/
    |-- element/string.json
    `-- media/
        |-- layered_image.json
        |-- background.png
        `-- foreground.png
```

- `AppScope/app.json5`：应用级 bundle、版本、全局图标和全局 label 配置。
- `AppScope/resources/base/element/string.json`：应用显示名，当前 `app_name` 为 `蹦蹦单词`。
- `AppScope/resources/base/media/layered_image.json`：应用分层图标配置，组合 `background.png` 与 `foreground.png`。
- `AppScope/resources/base/media/foreground.png`：由 Figma `01_app_icon 1` 导出的开心 Lexi 图标前景图。

### 2.2 Entry HAP

```text
products/bongbongword/
|-- oh-package.json5
|-- hvigorfile.ts
`-- src/main/
    |-- ets/
    |   |-- bongbongwordability/BongbongwordAbility.ets
    |   |-- bongbongwordbackupability/BongbongwordBackupAbility.ets
    |   `-- pages/Index.ets
    |-- module.json5
    `-- resources/base/
        |-- element/string.json
        `-- media/
            |-- layered_image.json
            |-- foreground.png
            |-- startIcon.png
            `-- tab_home_active.png
```

- `products/bongbongword/src/main/ets/pages/Index.ets`：主页面入口，承载四 Tab 容器；首页接入 `HomeTabView`，练习、蹦蹦星球、我的为占位视图。
- `products/bongbongword/src/main/ets/bongbongwordability/BongbongwordAbility.ets`：主 `UIAbility`，设置颜色模式并加载 `pages/Index`。
- `products/bongbongword/src/main/ets/bongbongwordbackupability/BongbongwordBackupAbility.ets`：备份扩展 Ability。
- `products/bongbongword/src/main/module.json5`：Entry HAP 模块声明、Ability、备份扩展、入口图标与启动背景配置。
- `products/bongbongword/src/main/resources/base/element/string.json`：Ability label，当前 `BongbongwordAbility_label` 为 `蹦蹦单词`。
- `products/bongbongword/src/main/resources/base/media/foreground.png`：Entry 分层图标前景图，来源为 Figma `01_app_icon 1`。
- `products/bongbongword/src/main/resources/base/media/startIcon.png`：启动窗口图标，来源为开心 Lexi 图标资源。
- `products/bongbongword/src/main/resources/base/media/tab_home_active.png`：主 Tab 首页选中态 icon，供 Entry 内 `Index.ets` 直接引用。
- `products/bongbongword/oh-package.json5`：Entry 模块依赖 `@bongbong/home` 与 `@bongbong/basic`。

### 2.3 公共模块 commons/basic

```text
commons/basic/
|-- Index.ets
`-- src/main/ets/
    |-- components/
    |   |-- EnergyProgress.ets
    |   |-- GlassCard.ets
    |   |-- PlaceholderTabView.ets
    |   `-- SectionHeader.ets
    |-- theme/BongBongTheme.ets
    |-- pages/Index.ets
    `-- utils/Calc.ets
```

- `commons/basic/Index.ets`：公共模块导出口，集中导出主题 token、基础组件和工具函数。
- `commons/basic/src/main/ets/theme/BongBongTheme.ets`：统一颜色、字号、圆角、阴影、间距、透明度等视觉 token。
- `commons/basic/src/main/ets/components/EnergyProgress.ets`：元气能量进度条组件，内置百分比边界保护。
- `commons/basic/src/main/ets/components/GlassCard.ets`：白色半透明卡片容器，统一卡片圆角、描边和阴影。
- `commons/basic/src/main/ets/components/SectionHeader.ets`：区块标题行，包含橙色圆点、标题和可选操作文本。
- `commons/basic/src/main/ets/components/PlaceholderTabView.ets`：未完成 Tab 的占位页。
- `commons/basic/src/main/ets/pages/Index.ets`、`utils/Calc.ets`：模板遗留示例文件，当前不承载核心业务。

### 2.4 首页模块 features/home

```text
features/home/
|-- Index.ets
`-- src/main/
    |-- ets/
    |   |-- views/HomeTabView.ets
    |   |-- components/
    |   |   |-- HomeEnergyCard.ets
    |   |   |-- TaskSummaryCard.ets
    |   |   |-- WordBookCarousel.ets
    |   |   `-- CheckInVine.ets
    |   |-- constants/
    |   |   |-- HomeMockData.ets
    |   |   `-- HomeMedia.ets
    |   |-- model/HomeModels.ets
    |   |-- viewmodel/HomeViewModel.ets
    |   |-- pages/Index.ets
    |   `-- utils/Calc.ets
    `-- resources/base/media/
        |-- home_lexi_hero.png
        |-- home_message_icon.png
        |-- home_energy_icon.png
        `-- tab_home_active.png
```

- `features/home/Index.ets`：首页模块导出口，当前导出 `HomeTabView`。
- `features/home/src/main/ets/views/HomeTabView.ets`：首页整体视图，组织顶部问候、Figma 消息 icon、开心 Lexi 主视觉、元气卡、任务卡、词书轮播和连续签到；当前已压缩纵向间距，并让能量卡与主视觉形成更接近设计稿的层叠关系。
- `features/home/src/main/ets/components/HomeEnergyCard.ets`：元气能量卡片，使用 `home_energy_icon.png`，保留 ArkUI 原生文本与 `EnergyProgress` 进度条。
- `features/home/src/main/ets/components/TaskSummaryCard.ets`：今日任务卡片，展示完成环形进度、任务列表和底部进度条。
- `features/home/src/main/ets/components/WordBookCarousel.ets`：我的词书横向轮播卡片。
- `features/home/src/main/ets/components/CheckInVine.ets`：连续签到藤蔓组件。
- `features/home/src/main/ets/model/HomeModels.ets`：首页静态数据模型，包括任务、词书、签到节点和首页状态。
- `features/home/src/main/ets/constants/HomeMockData.ets`：首页当前静态 mock 数据。
- `features/home/src/main/ets/constants/HomeMedia.ets`：首页图片资源引用集中管理，当前引用 Figma 首页切片。
- `features/home/src/main/resources/base/media/home_lexi_hero.png`：Figma `04_home_ui_concept 1` 裁切出的开心 Lexi 主视觉；不包含能量卡文字和进度条。
- `features/home/src/main/resources/base/media/home_message_icon.png`：Figma 首页消息 icon 切片。
- `features/home/src/main/resources/base/media/home_energy_icon.png`：Figma 首页能量闪电 icon 切片。
- `features/home/src/main/resources/base/media/tab_home_active.png`：Figma 底部导航首页 active icon 切片。
- `features/home/src/main/ets/pages/Index.ets`、`utils/Calc.ets`：模板遗留示例文件，当前不承载核心首页流程。

## 3. 当前已实现功能

- App 名称：桌面应用名与 Ability label 已统一为 `蹦蹦单词`。
- App 图标：应用级和 Entry 级图标已改为开心 Lexi 图标资源。
- 主入口：`BongbongwordAbility` 加载 `pages/Index`。
- 主 Tab：四个底部 Tab，分别是首页、练习、蹦蹦星球、我的；首页 active icon 使用 Figma 切片，其他 Tab 暂保留 ArkUI 文本图标，避免接入不干净的裁片。
- 首页：静态首页仪表盘，包含问候语、消息入口、开心 Lexi 主视觉、元气能量、今日任务、我的词书、连续签到。
- 首页布局：顶部区域按设计稿方向调整为问候语 / 消息按钮在上，Lexi 居中，元气卡压到主视觉下方；整体纵向间距已从偏松散状态收紧。
- 首页状态：通过 `HomeViewModel` 返回 `HOME_DASHBOARD_STATE` 静态数据，页面层不直接拼业务数据。
- 公共视觉：颜色、字号、间距、圆角、阴影等统一放在 `BongBongTheme.ets`。
- 公共组件：`GlassCard`、`EnergyProgress`、`SectionHeader`、`PlaceholderTabView` 已在首页和主 Tab 中复用。
- 词灵方案：`PET_SHAPE_PLAN.md` 已明确 Lexi 是学习伙伴，不做强养成惩罚型宠物系统。

### 4.3 资源规则

- 不把整张 Figma 首页稿挂到页面上；页面仍由 ArkUI 组件搭结构。
- 只提交语义化切片资源，不提交 `.tmp`、Figma 验证截图或临时裁切图。
- `02_lexi_character 1` 只作为角色参考，不再把旧横版角色设定图作为首页主视觉。
- 非首页 Tab 的 Figma 裁片当前不够干净，暂不接入；后续等独立 icon 层或更干净切片后再替换。
- 新增资源命名使用语义化小写下划线，例如 `home_lexi_hero.png`、`home_message_icon.png`。
