# BongBongWord 项目结构与功能设计

更新时间：2026-05-25

## 1. 项目定位

`BongBongWord` 是面向 HarmonyOS 6.0.2(22) 的 ArkTS / ArkUI 单词学习应用。当前阶段以 V2 第一阶段静态首页和主 Tab 骨架为主，工程采用 `products + features + commons` 分层：

```text
BongBongWord/
|-- AppScope/                  # 应用级配置和图标资源
|-- commons/
|   `-- basic/                 # 公共 HSP：主题、基础组件、通用工具
|-- features/
|   |-- home/                  # 首页 HSP：首页仪表盘与首页组件
|   |-- login/                 # 登录 HSP：当前为模板占位
|   |-- mine/                  # 我的 HSP：当前为模板占位
|   |-- pet/                   # 词灵伙伴 HSP：当前为模板占位，已有 Pet 方案文档
|   `-- practice/              # 练习 HSP：当前为模板占位
|-- products/
|   `-- bongbongword/          # Entry HAP：应用入口、Ability、主 Tab 容器
|-- hvigor/                    # Hvigor 工程配置
|-- build-profile.json5        # 工程模块与 SDK 配置
|-- oh-package.json5           # 根依赖声明
`-- PET_SHAPE_PLAN.md          # Lexi / Pet 形态方案
```

## 2. 关键目录与文件注释

### 2.1 Entry HAP

```text
products/bongbongword/
|-- oh-package.json5
|-- hvigorfile.ts
`-- src/main/
    |-- ets/
    |   |-- bongbongwordability/BongbongwordAbility.ets
    |   |-- bongbongwordbackupability/BongbongwordBackupAbility.ets
    |   `-- pages/Index.ets
    |-- module.json5
    `-- resources/
```

- `products/bongbongword/src/main/ets/pages/Index.ets`：主页面入口，承载四 Tab 容器；当前接入 `HomeTabView`，练习、蹦蹦星球、我的为占位视图。
- `products/bongbongword/src/main/ets/bongbongwordability/BongbongwordAbility.ets`：主 `UIAbility`，设置颜色模式并加载 `pages/Index`。
- `products/bongbongword/src/main/ets/bongbongwordbackupability/BongbongwordBackupAbility.ets`：备份扩展 Ability。
- `products/bongbongword/src/main/module.json5`：Entry HAP 模块声明、Ability、备份扩展、入口图标与启动背景配置。
- `products/bongbongword/oh-package.json5`：Entry 模块依赖 `@bongbong/home` 与 `@bongbong/basic`。

### 2.2 公共模块 commons/basic

```text
commons/basic/
|-- Index.ets
`-- src/main/ets/
    |-- components/
    |   |-- EnergyProgress.ets
    |   |-- GlassCard.ets
    |   |-- PlaceholderTabView.ets
    |   `-- SectionHeader.ets
    |-- theme/BongBongTheme.ets
    |-- pages/Index.ets
    `-- utils/Calc.ets
```

- `commons/basic/Index.ets`：公共模块导出口，集中导出主题 token、基础组件和工具函数。
- `commons/basic/src/main/ets/theme/BongBongTheme.ets`：统一颜色、字号、圆角、阴影、间距、透明度等视觉 token。
- `commons/basic/src/main/ets/components/EnergyProgress.ets`：元气能量进度条组件，内置百分比边界保护。
- `commons/basic/src/main/ets/components/GlassCard.ets`：白色半透明卡片容器，统一卡片圆角、描边和阴影。
- `commons/basic/src/main/ets/components/SectionHeader.ets`：区块标题行，包含橙色圆点、标题和可选操作文本。
- `commons/basic/src/main/ets/components/PlaceholderTabView.ets`：未完成 Tab 的占位页。
- `commons/basic/src/main/ets/pages/Index.ets`、`utils/Calc.ets`：模板遗留示例文件，当前不承载核心业务。

### 2.3 首页模块 features/home

```text
features/home/
|-- Index.ets
`-- src/main/
    |-- ets/
    |   |-- views/HomeTabView.ets
    |   |-- components/
    |   |   |-- HomeEnergyCard.ets
    |   |   |-- TaskSummaryCard.ets
    |   |   |-- WordBookCarousel.ets
    |   |   `-- CheckInVine.ets
    |   |-- constants/
    |   |   |-- HomeMockData.ets
    |   |   `-- HomeMedia.ets

    |   |-- viewmodel/HomeViewModel.ets
    |   |-- pages/Index.ets
    |   `-- utils/Calc.ets
    `-- resources/base/media/
        `-- lexi_home.png
```

- `features/home/Index.ets`：首页模块导出口，当前导出 `HomeTabView`。
- `features/home/src/main/ets/views/HomeTabView.ets`：首页整体视图，组织顶部问候、Lexi 主视觉、元气卡、任务卡、词书轮播和连续签到。
- `features/home/src/main/ets/components/HomeEnergyCard.ets`：元气能量卡片，展示图标、文字、数值和进度条。
- `features/home/src/main/ets/components/TaskSummaryCard.ets`：今日任务卡片，展示完成环形进度、任务列表和底部进度条。
- `features/home/src/main/ets/components/WordBookCarousel.ets`：我的词书横向轮播卡片。
- `features/home/src/main/ets/components/CheckInVine.ets`：连续签到藤蔓组件。
- `features/home/src/main/ets/model/HomeModels.ets`：首页静态数据模型，包括任务、词书、签到节点和首页状态。
- `features/home/src/main/ets/constants/HomeMockData.ets`：首页当前静态 mock 数据。
- `features/home/src/main/ets/constants/HomeMedia.ets`：首页图片资源引用集中管理。
- `features/home/src/main/resources/base/media/lexi_home.png`：当前首页 Lexi 图片资源，来源为本地角色设定图；后续应替换为 Figma 首页主视觉切片。
- `features/home/src/main/ets/pages/Index.ets`、`utils/Calc.ets`：模板遗留示例文件，当前不承载核心首页流程。

### 2.4 其他 feature 模块

- `features/login`：登录模块 HSP，当前保留模板页与测试骨架；后续承接手机号验证码、微信、支付宝、华为登录。
- `features/practice`：练习模块 HSP，当前保留模板页与测试骨架；后续承接单词题卡、听音、答题和 Combo 反馈。
- `features/pet`：词灵伙伴模块 HSP，当前保留模板页与测试骨架；产品边界见 `PET_SHAPE_PLAN.md`。
- `features/mine`：我的模块 HSP，当前保留模板页与测试骨架；后续承接学习数据、成就墙和技能树。

## 3. 当前已实现功能

- 主入口：`BongbongwordAbility` 加载 `pages/Index`。
- 主 Tab：四个底部 Tab，分别是首页、练习、蹦蹦星球、我的。
- 首页：静态首页仪表盘，包含问候语、消息入口、Lexi 主视觉、元气能量、今日任务、我的词书、连续签到。
- 首页状态：通过 `HomeViewModel` 返回 `HOME_DASHBOARD_STATE` 静态数据，页面层不直接拼业务数据。
- 公共视觉：颜色、字号、间距、圆角、阴影等统一放在 `BongBongTheme.ets`。
- 公共组件：`GlassCard`、`EnergyProgress`、`SectionHeader`、`PlaceholderTabView` 已在首页和主 Tab 中复用。
- 词灵方案：`PET_SHAPE_PLAN.md` 已明确 Lexi 是学习伙伴，不做强养成惩罚型宠物系统。
